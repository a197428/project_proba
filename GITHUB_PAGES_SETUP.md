# Настройка GitHub Pages для просмотра калькулятора онлайн

## 🌐 Что такое GitHub Pages?

GitHub Pages позволяет хостить статические HTML файлы прямо из вашего репозитория, делая их доступными по красивой ссылке.

## 📝 Инструкция по настройке

### Шаг 1: Откройте настройки репозитория

1. Перейдите на GitHub: https://github.com/a197428/project_proba
2. Нажмите на вкладку **Settings** (Настройки)

### Шаг 2: Найдите раздел Pages

1. В левом меню найдите раздел **Pages**
2. Или перейдите напрямую: https://github.com/a197428/project_proba/settings/pages

### Шаг 3: Настройте источник

1. В разделе **Source** (Источник) выберите:
   - **Branch:** `main`
   - **Folder:** `/ (root)`
2. Нажмите **Save** (Сохранить)

### Шаг 4: Дождитесь развертывания

1. GitHub начнет развертывание (обычно 1-2 минуты)
2. Вы увидите сообщение: "Your site is live at https://a197428.github.io/project_proba/"

### Шаг 5: Проверьте ссылки

После развертывания ваши файлы будут доступны по адресам:

#### Основные ссылки:
- **Калькулятор:** https://a197428.github.io/project_proba/calculator-preview.html
- **Тесты:** https://a197428.github.io/project_proba/calculator-tests.html

#### Документация:
- **Главная:** https://a197428.github.io/project_proba/
- **Быстрый старт:** https://a197428.github.io/project_proba/QUICK_START.md
- **Примеры:** https://a197428.github.io/project_proba/DEMO_EXAMPLES.md

## 🔄 Альтернативный способ (без настройки)

Если не хотите настраивать GitHub Pages, используйте сервис htmlpreview:

### Калькулятор:
```
https://htmlpreview.github.io/?https://github.com/a197428/project_proba/blob/main/calculator-preview.html
```

### Тесты:
```
https://htmlpreview.github.io/?https://github.com/a197428/project_proba/blob/main/calculator-tests.html
```

**Преимущества htmlpreview:**
- ✅ Работает сразу, без настройки
- ✅ Не требует прав администратора

**Недостатки:**
- ⚠️ Может быть медленнее
- ⚠️ Длинные URL

Если у вас есть свой домен, вы можете настроить его в разделе **Custom domain** в настройках Pages.

## 🔧 Устранение проблем

### Проблема: Сайт не открывается

**Решение:**
1. Проверьте, что GitHub Pages включен в настройках
2. Дождитесь завершения развертывания (зеленая галочка в Actions)
3. Очистите кеш браузера (Ctrl+Shift+R)

### Проблема: Файлы не обновляются

**Решение:**
1. Убедитесь, что изменения закоммичены и запушены
2. Дождитесь нового развертывания (1-2 минуты)
3. Очистите кеш браузера

### Проблема: 404 ошибка

**Решение:**
1. Проверьте правильность URL
2. Убедитесь, что файл существует в репозитории
3. Проверьте регистр букв в имени файла
