# Демонстрация возможностей калькулятора

## 🎯 Проблема с плавающей точкой - РЕШЕНА!

### Классическая проблема JavaScript

В обычном JavaScript:
```javascript
0.1 + 0.2 = 0.30000000000000004  ❌ НЕПРАВИЛЬНО!
```

### Наш калькулятор с Decimal.js:
```
0.1 + 0.2 = 0.3  ✅ ПРАВИЛЬНО!
```

## 📊 Примеры точных вычислений

### 1. Финансовые расчеты

**Пример: Расчет НДС**
```
Сумма без НДС: 1000
НДС 20%: 1000 × 0.2 = 200
Итого: 1000 + 200 = 1200
```

**Пример: Расчет скидки**
```
Цена: 2500
Скидка 15%: 2500 × 0.15 = 375
Итоговая цена: 2500 − 375 = 2125
```

### 2. Научные вычисления

**Пример: Площадь круга**
```
Радиус: 5
π ≈ 3.14159265
Площадь: 3.14159265 × 5 × 5 = 78.5398163
```

**Пример: Скорость**
```
Расстояние: 150 км
Время: 2.5 часа
Скорость: 150 ÷ 2.5 = 60 км/ч
```

### 3. Повседневные расчеты

**Пример: Разделить счет**
```
Общая сумма: 1234.56
Количество человек: 4
На каждого: 1234.56 ÷ 4 = 308.64
```

**Пример: Конвертация валют**
```
Сумма в долларах: 100
Курс: 75.5
В рублях: 100 × 75.5 = 7550
```

### 4. Периодические дроби

**1/3 с точностью до 10 знаков**
```
1 ÷ 3 = 0.3333333333
```

**2/3 с точностью до 10 знаков**
```
2 ÷ 3 = 0.6666666667
```

**1/7 с точностью до 10 знаков**
```
1 ÷ 7 = 0.1428571429
```

### 5. Последовательные операции

**Пример: Сложный расчет**
```
Шаг 1: 100 + 50 = 150
Шаг 2: × 2 = 300
Шаг 3: − 50 = 250
Шаг 4: ÷ 5 = 50
```

**Пример: Проценты от процентов**
```
Начальная сумма: 1000
+10%: 1000 × 1.1 = 1100
+5%: 1100 × 1.05 = 1155
```

### 6. Отрицательные числа

**Температурные расчеты**
```
Утром: -5°C
Потеплело на: 8°C
Днем: -5 + 8 = 3°C
```

**Финансы (убытки/прибыль)**
```
Убыток: -1000
Прибыль: 1500
Итого: -1000 + 1500 = 500
```

### 7. Очень маленькие числа

**Микрометры**
```
0.0001 + 0.0002 = 0.0003
```

**Проценты**
```
0.001 × 100 = 0.1
```

### 8. Большие числа

**Миллионы**
```
1000000 + 2000000 = 3000000
```

**Миллиарды**
```
999999999999 + 1 = 1000000000000
```

## 🎓 Учебные примеры

### Математика 5 класс

**Дроби**
```
1/2 = 0.5
1/4 = 0.25
3/4 = 0.75
```

**Проценты**
```
50% от 200 = 200 × 0.5 = 100
25% от 80 = 80 × 0.25 = 20
```

### Математика 6 класс

**Пропорции**
```
Если 3 яблока стоят 150 рублей,
то 1 яблоко стоит: 150 ÷ 3 = 50 рублей
```

**Масштаб**
```
Масштаб 1:100
Реальная длина: 5 метров = 500 см
На карте: 500 ÷ 100 = 5 см
```

### Физика

**Скорость, время, расстояние**
```
v = s / t
s = 120 км, t = 2 часа
v = 120 ÷ 2 = 60 км/ч
```

**Плотность**
```
ρ = m / V
m = 500 г, V = 250 см³
ρ = 500 ÷ 250 = 2 г/см³
```
