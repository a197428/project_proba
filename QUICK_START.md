# Быстрый старт - Личный калькулятор

## 🚀 Запуск за 3 шага

### 1. Откройте калькулятор
Откройте файл **`calculator-preview.html`** в браузере (двойной клик или перетащите в окно браузера)

### 2. Начните вычисления
Используйте мышь или клавиатуру для ввода:
- **Цифры**: 0-9
- **Операции**: + − × ÷
- **Результат**: = или Enter
- **Очистка**: C или Escape

### 3. Проверьте точность (опционально)
Откройте **`calculator-tests.html`** для запуска автоматических тестов

## ✨ Ключевые особенности

### Высокая точность
```
0.1 + 0.2 = 0.3  ✓ (точный результат!)
1 ÷ 3 = 0.3333333333  ✓ (10 знаков после запятой)
```

### Последовательные вычисления
```
5 + 3 = 8
× 2 = 16
÷ 4 = 4
```

### Повторение операции
```
10 + 5 = 15
= 20  (повторяет +5)
= 25  (повторяет +5)
```

## 🎨 Современный дизайн

- Темная цветовая схема
- Плавные анимации при нажатии
- Адаптивный интерфейс для всех устройств

## ⌨️ Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| **0-9** | Ввод цифр |
| **+ - * /** | Операции |
| **Enter** | Вычислить |
| **Escape** | Очистить |
| **Backspace** | Удалить символ |
| **.** | Десятичная точка |

## ❓ Проблемы?

1. **Калькулятор не загружается?**
   - Проверьте подключение к интернету (нужно для Decimal.js)
   - Используйте современный браузер

2. **Неточные результаты?**
   - Запустите тесты в `calculator-tests.html`
   - Все тесты должны пройти успешно

3. **Нужна помощь?**
   - Откройте `CALCULATOR_README.md` для подробной документации
