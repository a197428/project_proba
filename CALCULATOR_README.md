# Личный калькулятор с высокой точностью

Полнофункциональный веб-калькулятор с современным дизайном и высокоточными вычислениями.

## Особенности

✅ **Высокая точность вычислений** - использует библиотеку Decimal.js для точных вычислений без ошибок округления с плавающей точкой

✅ **Современный дизайн** - темная цветовая схема, плавные анимации, адаптивный интерфейс

✅ **Поддержка клавиатуры** - полная поддержка ввода с клавиатуры для быстрой работы

✅ **Все базовые операции** - сложение, вычитание, умножение, деление

✅ **Последовательные вычисления** - возможность выполнять цепочки операций

✅ **Обработка ошибок** - корректная обработка деления на ноль и других граничных случаев

### Локальный запуск

1. Откройте файл `calculator-preview.html` в любом современном браузере
2. Калькулятор готов к использованию!

### Управление мышью

- Нажимайте на кнопки с цифрами (0-9) для ввода чисел
- Нажимайте на кнопки операций (+, −, ×, ÷) для выбора операции
- Нажмите "=" для получения результата
- Нажмите "C" для очистки текущего ввода
- Нажмите "AC" для полной очистки калькулятора
- Нажмите "⌫" для удаления последнего символа
- Нажмите "." для ввода десятичной точки

### Управление клавиатурой

| Клавиша | Действие |
|---------|----------|
| 0-9 | Ввод цифр |
| + | Сложение |
| - | Вычитание |
| * | Умножение |
| / | Деление |
| . или , | Десятичная точка |
| Enter или = | Вычислить результат |
| Escape или C | Очистить текущий ввод |
| Backspace | Удалить последний символ |

## Примеры использования

### Простое вычисление
```
5 + 3 = 8
```

### Последовательные операции
```
10 + 5 = 15
× 2 = 30
÷ 3 = 10
```

### Десятичные числа
```
0.1 + 0.2 = 0.3  (точный результат!)
```

### Повторение операции
```
5 + 3 = 8
= 11  (повторяет +3)
= 14  (повторяет +3)
```

## Проверка точности

Откройте файл `calculator-tests.html` в браузере для запуска автоматических тестов точности вычислений.

Тесты проверяют:
- Базовые арифметические операции
- Вычисления с десятичными числами
- Проблемные случаи с плавающей точкой (0.1 + 0.2)
- Большие числа
- Отрицательные числа
- Деление на ноль
- Точность округления до 10 знаков после запятой
- Периодические дроби

## Технические детали

### Архитектура

Калькулятор построен по паттерну MVC:

- **CalculatorEngine** - вычислительный движок с логикой операций
- **CalculatorUI** - управление отображением и анимациями
- **CalculatorController** - обработка пользовательского ввода

### Библиотеки

- **Decimal.js** - библиотека для высокоточных вычислений с произвольной точностью

### Точность

- Вычисления выполняются с точностью до 10 знаков после запятой
- Результаты автоматически округляются по правилам математического округления
- Длинные числа (>15 символов) отображаются в научной нотации

### Ограничения

- Максимальная длина ввода: 15 цифр
- Точность результата: 10 знаков после запятой
- Поддерживаемые операции: +, −, ×, ÷

## Решение проблем

### Калькулятор не работает

- Убедитесь, что у вас есть подключение к интернету (для загрузки Decimal.js)
- Используйте современный браузер (Chrome, Firefox, Safari, Edge)
- Проверьте консоль браузера на наличие ошибок (F12)

### Неточные результаты

- Калькулятор использует Decimal.js для точных вычислений
- Запустите `calculator-tests.html` для проверки точности
- Все тесты должны пройти успешно
