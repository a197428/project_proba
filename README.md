# project_proba

## 🧮 Личный калькулятор с высокой точностью

Полнофункциональный веб-калькулятор с современным дизайном и высокоточными вычислениями на основе Decimal.js.

### 🚀 Быстрый старт

1. **Откройте калькулятор**: `calculator-preview.html`
2. **Запустите тесты**: `calculator-tests.html`
3. **Читайте документацию**: `QUICK_START.md`

### 📁 Файлы проекта

| Файл | Описание |
|------|----------|
| `calculator-preview.html` | Основной файл калькулятора |
| `calculator-tests.html` | Автоматические тесты (30+ тестов) |
| `QUICK_START.md` | Быстрое руководство по использованию |
| `CALCULATOR_README.md` | Полная документация |
| `DEMO_EXAMPLES.md` | Примеры использования и демонстрация |

### ✨ Основные возможности

- ✅ **Высокая точность** - решена проблема 0.1 + 0.2 = 0.3
- ✅ **Современный дизайн** - темная тема, анимации, адаптивность
- ✅ **Поддержка клавиатуры** - полный контроль с клавиатуры
- ✅ **Все операции** - сложение, вычитание, умножение, деление
- ✅ **Последовательные вычисления** - цепочки операций
- ✅ **Обработка ошибок** - деление на ноль и граничные случаи

### 🎯 Примеры

```
0.1 + 0.2 = 0.3  ✓ (точный результат!)
1 ÷ 3 = 0.3333333333  ✓ (10 знаков)
100 + 50 × 2 − 25 = последовательные операции
```
