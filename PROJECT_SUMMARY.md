## 🏗️ Архитектура

### Паттерн: MVC (Model-View-Controller)

**Model (CalculatorEngine)**
- Хранение состояния калькулятора
- Вычислительная логика
- Обработка операций

**View (CalculatorUI)**
- Управление DOM
- Обновление дисплея
- Анимации

**Controller (CalculatorController)**
- Обработка событий
- Координация Model и View
- Клавиатурный ввод

### Компоненты

1. **CalculatorEngine** - вычислительный движок
2. **CalculatorUI** - пользовательский интерфейс
3. **CalculatorController** - контроллер событий
4. **NumberFormatter** - форматирование чисел (встроен в Engine)
5. **InputValidator** - валидация ввода (встроен в Controller)

## 🔧 Технические детали

### Используемые технологии
- **HTML5** - структура страницы
- **CSS3** - стилизация и анимации
- **JavaScript ES6+** - логика приложения
- **Decimal.js v10.4.3** - высокоточные вычисления

### Ключевые решения

1. **Decimal.js вместо Number**
   - Решает проблему 0.1 + 0.2
   - Обеспечивает точность до 10 знаков
   - Поддержка больших чисел

2. **Хранение чисел как строк**
   - Сохранение точности ввода
   - Избежание преждевременного округления
   - Контроль форматирования

3. **MVC архитектура**
   - Разделение ответственности
   - Легкость тестирования
   - Поддерживаемость кода

4. **Обработка событий**
   - Единый обработчик для кнопок
   - Отдельный обработчик для клавиатуры
   - Делегирование событий
